# Applehail\Webutils

Utility functions for simple sites

## Setup

    composer require applehail/webutils

## Functions

    config() - zend-config
    logger() - monolog
    starts_with($haystack, $needle)
    ends_with($haystack, $needle)
    d() - dump
    dd() - dump and die
    get_tag()
    get_tag_regexp()
    get_tag_array()
    filecache()
    translit()
    get_cpu()
